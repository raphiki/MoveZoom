MoveZoom

What is it?

MoveZoom is a small JavaScript Library allowing to zoom a portion of image by moving the move over it, as shown above.
  • It is released under the GNU Public License, any contribution, enhancement or adaptation is fairly welcomed.
  • Current version is 0.1, it can be downloaded here.
  • ScriptDoc Reference is available here.
  • It has been tested on Mozilla Firefox et Internet Explorer, please report any compatibility problems (raphael AT semeteys DOT org)

How to use it in your web page?

Check HTML/JavaScript code of this page...
  • Insert the MoveZoom.js script somewhere in your HTML document (in the <head> tag for instance): 
    <script type="text/javascript" src="MoveZoom.js"></script>
  • The image to become zoomable must have an unique id, width and height attributes can be used to reduce image's size. 
    <img id="myImage" src="rose.jpg" width="529" height="410"/>
  • Call the MoveZoom.init() function:
    • Either from the onload event of the HTML page or form another <script> insert (in that case it must be placed after the MoveZoom.js include)
    • With the image id and zoom ratio id as parameters 
      <body onload = "MoveZoom.init('myImage', 3);">
